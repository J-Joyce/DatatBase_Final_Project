# DatatBase_Final_Project People's Record

# Databases used: Microsoft Power BI Microsoft Power BI Ohio Population by Age - 2025 Update | Neilsberg 

# description:
    This app tracks voter registration and election data across Ohio. users can see how different populations across Ohio vote. Using the apps built in maps, users can get a better understanding of voter turnout and party affiliations statistics. The app will have data from multiple different elections allowing users to see how people voted in national and local elections.
